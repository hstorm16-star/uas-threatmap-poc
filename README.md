# UAS ThreatMap POC

AI-assisted OSINT and sanctions-oriented UAS procurement intelligence dashboard built with Streamlit, Python, and Plotly.

This project is a proof-of-concept analytical dashboard designed to model drone procurement ecosystems, sanctions adaptation behaviors, logistics pathways, and intelligence collection gaps using open-source reporting, analytical assessment, and synthetic/demo data.

---

# Overview

The dashboard explores how commercially available and dual-use drone components can move through layered procurement, logistics, telecom, financial, and sanctions adaptation networks.

The project was built as an analyst-facing prototype to demonstrate:
- intelligence workflow design
- network visualization
- procurement risk modeling
- sanctions adaptation analysis
- OSINT enrichment concepts
- research gap identification
- AI-assisted analytical tooling

---

# Features

## Threat Overview
- Entity and risk-level distribution
- Geographic exposure indicators
- Procurement category breakdown
- Confidence and source-type filtering

## Components Intelligence
- Drone component tracking
- Procurement relevance
- Component categorization
- Risk-weighted visibility

## Network Analysis
- Supply-chain relationship mapping
- Telecom/connectivity indicators
- Maritime/AIS enrichment concepts
- Sanctions adaptation lifecycle modeling

## Trends & Time-Series
- Incident trend analysis
- Incident-type evolution
- Heightened activity indicators
- Regional activity changes

## Research Gaps
Structured analytical assessment of:
- weakly supported assumptions
- underdeveloped regions
- logistics blind spots
- sanctions evasion typologies
- telecom and payment infrastructure gaps

## Analyst Briefing Layer
- Operational summary generation
- Intelligence framing
- Collection-priority guidance

# Important Disclaimer

This dashboard contains:
- open-source reporting
- analytical assessment
- inferred relationships
- and synthetic/demo data

It is intended solely for:
- intelligence workflow prototyping
- OSINT analysis
- sanctions/export-control research
- and portfolio demonstration purposes

It is NOT:
- a production intelligence platform
- an attribution engine
- or a substitute for primary-source verification.

Some analytical relationships shown are modeled or inferred to demonstrate workflow concepts and do not represent verified operational intelligence.
